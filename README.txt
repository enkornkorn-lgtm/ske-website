เปิดดูเว็บ:
1. เปิดไฟล์ index.html ด้วย Chrome หรือ Edge

แก้ข้อความ:
- เปิด index.html ด้วย Notepad, VS Code หรือโปรแกรมแก้ข้อความ
- แก้ข้อความภาษาไทยตรงส่วนที่ต้องการ แล้วบันทึกไฟล์
- กลับไป refresh หน้าเว็บที่เปิดอยู่

แก้สี/ขนาด/ระยะห่าง:
- เปิด style.css
- สีหลักอยู่ด้านบนใน :root เช่น --black, --orange, --gold, --blue

แก้รูป:
- ไฟล์รูปอยู่ในโฟลเดอร์ assets
- ถ้าจะเปลี่ยน hero ให้ใส่รูปใหม่ใน assets แล้วแก้บรรทัดนี้ใน index.html:
  assets/hero-sharp.jpg

หมายเหตุ:
- ชุดนี้เป็นไฟล์สำหรับตรวจ/แก้ไขในเครื่องก่อน ยังไม่อัปขึ้นเว็บจริง
- เมื่อแก้จนพอใจแล้ว ส่งไฟล์หรือบอกจุดที่แก้ให้ Codex อัปเดตเว็บจริงให้
